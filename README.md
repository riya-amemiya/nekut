# プログラミング学習サイト　Nekut

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/Z8Z3GGLNZ)

Nekutは無料でプログラミング学習を行うことができるサイトです。
テキストとYouTubeを組み合わせて学習を行うことができます。
チャプターごとにソースコードをダウンロードすることができます。
