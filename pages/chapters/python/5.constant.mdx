# Python の定数

今回のソースコードは<GetCodeLink>こちら</GetCodeLink>からダウンロードできます。

Python には定数という概念はありません。定数というのは、値を変更できない変数のことです。<br/>
Python では、定数を表すには、一般的に変数名を大文字で表し、変数名を大文字で表すことで、定数としての意図を示します。<br/>

そのため、定数を表す場合は、通常以下のようにします。

```python
CONSTANT_NAME = value
```

ただし、Python には、変数名を大文字で表したものを定数として扱うという慣習はありますが、Python 本来の機能として、変数を定数として扱うことはできません。<br/>
変数を定数として扱うには、プログラマが意図的に、その変数を変更しないようにする必要があります。<br/>
また、Python には、定数を表すためのライブラリがあります。例えば、「constants」や「enum」などがあります。<br/>
これらのライブラリを使用することで、より厳密な定数を表すことができます。
