# Python の変数

今回のソースコードは<GetCodeLink>こちら</GetCodeLink>からダウンロードできます。

Python で変数を宣言するには、変数名を決めて、その変数に値を代入することで宣言することができます。<br/>
例えば、以下のようになります。

```python
x = 10
y = "Hello, World!"
z = 3.14
```

この例では、変数 x に整数の 10、変数 y に文字列の "Hello, World!"、変数 z に浮動小数点数の 3.14 を代入しています。<br/>
Python では、変数を宣言すると同時に値を代入する必要があります。変数を宣言しただけでは、その変数は未定義の状態になります。<br/>
また、Python では、変数名は英数字とアンダースコアで構成されますが、変数名は数字から始めることができません。<br/>
また、予約語や Python のモジュール名などを変数名に使用することはできませんので、注意する必要があります。
