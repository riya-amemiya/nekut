# C 言語の定数

今回のソースコードは<GetCodeLink>こちら</GetCodeLink>からダウンロードできます。

C 言語では、定数は文字列や数値などを定数名に対応させることができます。<br/>
定数は、宣言された値を変更することができないため、プログラムの中で値が変化しないような値に使用されることがよくあります。<br/>
また、定数は、大文字で書かれることが多く、定数名を宣言する際には、アンダースコアを使用することがよくあります。<br/>
これは、定数名を区別しやすくするためです。<br/>
C 言語では const キーワードを使用することでも宣言することができます。例えば、以下のようになります。

```c
const double PI = 3.14;
const int MAX_VALUE = 100;
```

Google C++ Style Guide では、定数は、k で始まる名前を使用することを推奨しています。<br/>
これは、定数名を区別しやすくするためです。<br/>

```c
const double kPi = 3.14;
const int kMaxValue = 100;
```
