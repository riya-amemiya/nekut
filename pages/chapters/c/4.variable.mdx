# C 言語の変数

今回のソースコードは<GetCodeLink>こちら</GetCodeLink>からダウンロードできます。

C 言語で変数を宣言するには、変数名と型を指定して宣言する必要があります。例えば、以下のようになります。<br/>

```c
int x;
x = 10;
char y;
y = 'A';
double z;
z = 3.14;
```

この例では、変数 x を整数型の変数、変数 y を文字型の変数、変数 z を浮動小数点型の変数として宣言しています。<br/>
C 言語では、変数を宣言すると同時に値を代入することもできます。例えば、以下のようになります。

```c
int x = 10;
char y = 'A';
double z = 3.14;
```

C 言語では、変数名は英数字とアンダースコアで構成されますが、変数名は数字から始めることができません。<br/>
また、予約語を変数名に使用することはできませんので、注意する必要があります。

## 基本の型

C 言語では、以下のような基本の型があります。

-   int : 整数型 (4 バイト) 0 から 2,147,483,647 までの値を格納できます。
-   char : 文字型 (1 バイト) 0 から 255 までの値を格納できます。
-   double : 浮動小数点型 (8 バイト) 1.7E-308 から 1.7E+308 までの値を格納できます。
-   float : 浮動小数点型 (4 バイト) 1.2E-38 から 3.4E+38 までの値を格納できます。
-   void : 型なし
-   bool : 論理型 (1 バイト) true または false の値を格納できます。

その他の型は他の章で説明します。
