# JavaScript の変数

今回のソースコードは<GetCodeLink>こちら</GetCodeLink>からダウンロードできます。

JavaScript の変数は、値を格納するための箱のようなものです。<br/>
変数には、値を格納することができます。

```js
let age = 18;
```

変数には分かりやすい名前をつけることが望ましいです。

```js
let x = 18; // なんの値かわからない
let age = 18; // なんの値かわかる
```

変数には、値を再代入することができます。

```js
let age = 18;
age = 19;
```
